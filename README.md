# weather-forecast
Weather Application 🌤️

A dynamic weather web application that allows users to search for any city and view real-time weather information along with upcoming 3 days forecast data. The interface updates visually based on the current weather conditions, providing an engaging and interactive user experience. The application integrates weather and image APIs to display relevant city imagery and weather data, creating a more immersive interface for users.

🕸️Features

-Search for weather by city name

-Displays current temperature and weather conditions

-Shows forecast for upcoming days

-Dynamic background changes depending on weather conditions (rain, sunny, cloudy, etc.)

-Displays city-related images based on the user’s search

-Loading animation while data is being fetched

-Responsive design

⛏️Technologies Used

-HTML

-CSS

-JavaScript

-Weather API integration

-Pexels API for city imagery

🎇Concepts Practiced

This project demonstrates:

-API integration (RESTful API)

-Asynchronous JavaScript (fetch, async/await)

-DOM manipulation

-Dynamic UI updates

-Conditional rendering based on API data

🔮Future Improvements

-Add hourly weather forecast

-Add geolocation to detect the user's current city

-Improve UI animations and transitions

-Allow users to switch between Celsius and Fahrenheit

-Cache previously searched cities
